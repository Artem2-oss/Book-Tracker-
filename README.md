# Book Tracker — Трекер прочитанных книг

**Автор:** Максимча Артем

## Описание программы

Book Tracker — это десктопное приложение с графическим интерфейсом (Python + Tkinter) для ведения списка прочитанных книг.  
Позволяет:

- Добавлять книгу (название, автор, жанр, кол-во страниц)
- Просматривать список в удобной таблице
- Фильтровать по жанру (частичное совпадение) и по минимальному количеству страниц
- Сохранять все данные в JSON-файл и загружать из него
- Валидация ввода: поля не пустые, страницы — целое число

## Инструкция по запуску

1. Убедитесь, что установлен Python 3.6+
2. Скачайте файл `book_tracker.py`
3. Запустите в терминале:
   ```bash
   python book_tracker.py
